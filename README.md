# 🎲 Dice Game

A simple, interactive, and responsive dice game built with **HTML**, **CSS**, and **JavaScript**.

## 🚀 Live Demo

👉 **Play here:** https://behrokh-dev.github.io/Dice-game/

## 📸 Preview



## ✨ Features

- 🎲 Random dice roll generation
- 👥 Two-player gameplay
- 🏆 Automatic winner detection
- 🔄 Play again by refreshing the page
- 📱 Responsive design
- 🎨 Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 💡 Future Improvements

- Add a **Roll Dice** button instead of refreshing the page.
- Keep track of player scores.
- Add dice rolling animations.
- Include sound effects.
- Support multiple rounds.
- Add dark mode.
